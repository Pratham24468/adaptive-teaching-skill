# adaptive-teaching-skill
adaptive-teaching Skill — What It Does
Trigger Conditions
Any frontier AI model with this skill installed will activate it whenever a user says anything related to learning — "teach me", "explain this", "I don't get it", "help me understand", "walk me through", etc.
Core Behavior (Research-Grounded)
The skill enforces a 7-step teaching protocol drawn directly from your Research.md:

Diagnose first — Identify the learner's prior knowledge and whether they're analytical (Field-Independent) or contextual (Field-Dependent) before teaching a single thing
Activate prior knowledge — Connect to what they already know before introducing new concepts
Worked example — Always show a fully solved/narrated example before asking the learner to attempt anything (especially for novices)
Faded scaffolding — Guide through a problem, then pull support back progressively
Retrieval practice — Never ask "does that make sense?" — always trigger active recall instead
Specific feedback — Targeted, error-correcting, explains the why behind mistakes
Metacognitive prompt — Ends every segment with "why does this work?" / "where would this fail?" to build transfer

What Makes It Unique

Covers all 5 domain types (abstract, procedural, factual, creative, social) with specific strategies for each
Has a hard block against learning style myths (no VARK)
Includes a "do NOT do this" table so the model avoids the most common bad teaching habits
The references/research-base.md bundles all effect sizes, domain roadmaps, and the full theory comparison for deeper context
