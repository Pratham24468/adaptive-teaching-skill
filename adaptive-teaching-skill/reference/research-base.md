## 1. Top Two Techniques (The Non-Negotiables)

A 2021 meta-analysis of 242 studies (169,179 participants) confirmed two techniques stand above all others:

### Retrieval Practice (Practice Testing)
- Effect size: 0.70–1.0+
- Mechanism: Effortful retrieval strengthens memory traces (long-term potentiation). The act of *trying to recall* is what encodes — not the re-reading of information.
- Application: Low-stakes quizzes, self-explanation, "without looking, explain X", flashcards
- Works universally; strongest effect for novices and lower-prior-knowledge learners

### Distributed / Spaced Practice
- Effect size: 0.70–1.0+
- Mechanism: Spacing forces re-encoding as memories fade slightly, making each retrieval more effortful and durable
- Application: Space sessions across days/weeks; expanding intervals (1 day → 3 days → 7 days → 30 days)
- Works for all ages, all domains, all learner profiles

**Key insight**: Both techniques work precisely *because* they are harder. Struggle is the signal that deep encoding is occurring. Easy, fluent review creates illusions of competence.

## 2. Learning Theories Compared

| Theory | Core Claim | Strongest Application | Limitation |
|--------|------------|----------------------|------------|
| **Behaviorism** (Skinner, Thorndike) | Learning = stimulus-response via reinforcement | Procedural automatization, habit formation | Shallow for conceptual depth |
| **Cognitivism / CLT** (Sweller 2024, Baddeley) | Working memory is severely limited; learning = schema construction | Novice instruction, reducing cognitive overload | Must adapt as expertise grows |
| **Constructivism** (Piaget, Vygotsky) | Learners actively build knowledge; ZPD with scaffolding | Guided inquiry, social learning | Fails novices without explicit support |
| **Neuroscience-based** | Neuroplasticity + long-term potentiation via spaced retrieval | Spacing, interleaving, sleep consolidation | Emerging; not all findings classroom-ready |
| **Flow Theory** (Csikszentmihalyi) | Optimal challenge-skill balance drives engagement | Calibrating difficulty to maintain engagement | Hard to operationalize precisely |

**Critical point (Kirschner/Sweller/Clark):** Pure discovery learning consistently fails novices. Constructivism only works *with* explicit guidance. Never leave a novice to figure it out alone.

## 3. Individual Cognitive Profiles

### Field-Independent (Analytical) — ~40–50% of population
- Process information by disembedding it from context; impose their own structure
- Strengths: math, science, independent problem-solving, abstract reasoning
- Memory preference: semantic and procedural
- Identification: fast on structured tasks, prefer internal referents, comfortable with ambiguity
- Best instruction: concept mapping, independent problems, advanced organizers, moderate-fast pacing with early structure → autonomy

### Field-Dependent (Holistic/Social) — complementary prevalence
- Process information contextually; rely on external structure; socially oriented
- Strengths: interpersonal learning, contextual pattern recognition, narrative understanding
- Memory preference: episodic and contextual
- Identification: prefers collaborative tasks, struggles with unstructured material
- Best instruction: collaborative learning, narrative/case studies, explicit scaffolding, social context in examples, slower pacing with more support

### Important: VARK "learning styles" (visual/auditory/kinesthetic) are NOT the same as FDI
- VARK has no empirical support; matching instruction to sensory preference produces no benefit
- FDI is measurable, valid, and has moderate but real implications for scaffolding design
- Use multimodal presentation universally — it benefits everyone

## 4. Teaching Methods Ranked by Effect Size

| Rank | Method | Effect Size | Best For |
|------|--------|-------------|---------|
| 1 | Distributed/Spaced Practice + Retrieval | 0.70–1.0+ | Universal — all learners, all domains |
| 2 | Direct/Explicit Instruction + Worked Examples | 0.59–0.82 | Novices; high cognitive-load topics |
| 3 | Specific, Timely Feedback | ~0.70 | Universal |
| 4 | Interleaved Practice | 0.46–0.59 | Universal; especially procedural/conceptual |
| 5 | Scaffolding + Faded Guidance | Moderate–High | Novice → intermediate transition |
| 6 | Collaborative/Peer Teaching (structured) | Moderate | Field-Dependent learners strongest |
| 7 | Metacognitive Prompting | Moderate | Transfer and self-regulation |
| 8 | Concept Mapping | Moderate | Declarative / relational knowledge |

**Contraindication**: Minimal guidance + discovery for novices → cognitive overload → failure. Always scaffold first.

## 5. Cognitive Science Mechanisms

### Working Memory (Sweller CLT 2024)
- Capacity: 4–7 chunks simultaneously
- Three types of load:
  - **Intrinsic**: complexity inherent to the material (can't eliminate, only sequence wisely)
  - **Extraneous**: caused by poor instruction design (eliminate this)
  - **Germane**: productive cognitive effort that builds schemas (maximize this)
- Design principles: integrated formats (no split attention), signaling, worked examples before problems, eliminate redundancy

### Long-Term Potentiation & Memory Consolidation
- Memories consolidate during sleep — spacing learning across days activates this
- Each retrieval re-encodes the memory more durably
- Interleaving forces the brain to discriminate between concepts, deepening encoding

### Metacognition & Self-Explanation
- Learners who explain their own reasoning outperform passive learners significantly
- Metacognitive prompts: "Why does this work?" "Where could this fail?" "Explain this without looking"
- Metacognition can be explicitly taught and transfers across domains

### Transfer of Learning
- Near transfer (same domain, different problem): achieved via varied examples
- Far transfer (different domain): requires explicit principle articulation + interleaving
- "Learn the rule, then see the rule across many contexts" is the transfer formula

### Neuroplasticity & Growth Mindset
- Brain malleability is real — abilities can grow with effort
- Growth mindset as an isolated intervention shows small effects (d ≈ 0.05) in rigorous studies
- More powerful when embedded in high-quality instruction + genuine challenge + specific feedback

## 6. Personalization & Adaptation

### Valid Approaches
- Prior-knowledge assessment (embedded short quiz or diagnostic question)
- Real-time cognitive load monitoring (learner confusion = overload signal)
- FDI-informed scaffolding (more structure for FD, earlier autonomy for FI)
- Adaptive spacing based on recall performance

### Invalid Approaches
- VARK questionnaires and sensory style matching (no evidence)
- Fixed ability labels ("she's just not a math person")
- Personality-based teaching assignments without evidence base

### Technology vs. Human Teaching
- AI tutors: excellent for spacing, retrieval pacing, immediate feedback, practice generation
- Human teachers: superior for metacognition scaffolding, motivation, relationships, and nuanced error diagnosis
- Best outcome: both, not either/or

## 7. Myths & Misconceptions

| Myth | Reality |
|------|---------|
| "Learning styles must be matched (VARK)" | No crossover interaction evidence; debunked across hundreds of studies |
| "Discovery learning is best" | Harms novices without scaffolding; effective only with explicit guidance |
| "Re-reading reinforces memory" | Creates fluency illusion; retrieval practice is vastly superior |
| "Growth mindset = achievement gains" | Small or null effects on grades in rigorous RCTs; framing alone is insufficient |
| "More content = more learning" | Cognitive overload reduces learning; less, spaced, is more |
| "Technology enhances learning automatically" | Only when designed around retrieval + feedback principles |
| "Confusion means bad teaching" | Desirable difficulty; effortful processing is the mechanism of deep encoding |

## 8. Domain-Specific Roadmaps

### Teaching Calculus (Abstract/Conceptual)
- Weeks 1–2: Worked examples of limits/derivatives with explicit narrated reasoning
- Ongoing: Spaced retrieval quizzes; interleave derivatives vs. integrals
- Metacognition: "Explain the rule in your own words. When does it not apply?"
- Assessment: Low-stakes practice tests → novel transfer problems

### Learning Coding (Procedural)
- Scaffolded worked examples → pair programming (FD-friendly) → independent projects
- Interleave syntax + logic + debugging
- Retrieval: "Explain this code without running it. What will the output be?"
- Generalize: "This pattern — where else have you seen it?"

### Teaching Communication Skills (Emotional/Social)
- Role-play + structured reflection after each scenario
- Video/peer feedback on delivery
- Spaced retrieval of frameworks ("Name the 3 steps of active listening — now apply one")
- Framing: "These skills grow with deliberate practice"

### Learning History/Law/Vocabulary (Declarative)
- Spaced flashcard-style retrieval built into sessions
- Organization schemes: "How does this connect to the Roman period we covered?"
- Expanding intervals: test today → 3 days → 1 week → 1 month
- Teach relationships, not isolated facts

### Teaching Writing/Design (Creative)
- Constraints first: "Write 200 words about X using only concrete nouns"
- Iteration cycle: draft → specific line-level feedback → revise
- Model excellence explicitly: "This sentence works because [specific reason]"
- Psychological safety: "Every first draft is supposed to be imperfect"
